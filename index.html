概率论和数理统计

发表于 2022-09-07 分类于 Math

不做数学上的推导，只进行基础知识的补充和知识体系的构架，抓住主干，忽略细枝末节

The Sample Space

Definition

  • The collection of all possible outcomes of an experiment.

Or

  • The sample space of an experiment is the set of all possible outcomes of that experiment.

一次试验的样本空间是该试验所有可能结果的集合。

Examples

  • Rolling a Die（掷色子）：投掷一个含六个面的色子，样本空间S={1,2,3,4,5,6}，每个数字表示色子一个可能的面。
  • 医院下一个出生的婴儿性别，S={F,M}
  • 扔一枚硬币朝上的面，S={H,T}
  • 连续扔三次硬币朝上的面，S={HHH,HHT,HTH,HTT,THH,THT,TTH,TTT}

Event

Definition

  • An event is any collection(subset) of outcomes contained in the sample space S. An event is said to be simple if it consists of exactly one outcome and compound if it consists of more than one outcom.
    事件是样本空间S的子集

Examples

  • 事件A：投掷一枚色子，正面朝上的数字是偶数，A={2,4,6}
  • 事件B：投掷一枚色子，正面朝上的数字大于3，B={4,5,6}

Probability

Definition

  • Given an experiment and a sample space S，the objective of probability is to assign to each event A a number P(A)，called the probability of the event A，which will give a precise measure of the chance that A will occur.
    给定一个试验和样本空间S，给每个事件A分配一个能准确度量其发生几率的数字P(A)，称为事件A的概率。

Or

  • A probability measure，or simply a probability，on a sample space S is a specification of numbers P(A) for all events A that satisfy 3 Axioms

Random Variables

Definition

  • For a given sample space S of some experiment，a random variable is any rule that associates a number with each outcome in S.

In general,each outcome of an experiment can be associate with a number by specifying a rule of association.Such a rule of association is called a random variable — a variable because different numerical values are possible and random because the observed value depends on which of the possible experimental outcomes results.
通常，一次试验的每个结果都可以通过一种特定的联合方式将该结果与数值结合，这种联合方式称为随机变量。

Examples

  • 一个同学欲使用学校公用电脑，有两种可能的结果：所有电脑都在使用（F），则该同学无法使用；至少有一台空闲（T），则同学可以使用。样本空间S={F,T}, 定义随机变量X，令X(T)=1; X(F)=0

                    # 一、文献检索与管理
                
                    ## 1. 文献的重要性
                    美国科学基金会统计，一个科研人员花费在查找和消化科技资料上的时间需占全部科研时间的 51%，计划思考占 8%，实验研究占 32%，书面总结占 9%。
                
                    
                
                       <!-- more -->          
                            
                    ## 2. 文献检索的基本步骤
                       如何查找参考文献      
                
                    - 推荐(导师、同学、参加会议与研讨会时结交的同领域人士)
                    - 关键词：topic,question,theory,.methodology
                    - 作者：领域有建树的学者（最有效），研究该领域前沿的科研者或实验室
                    - 领域：跨学科博览，扩展自己的知识体系       
                
                
                    如何获取参考文献全文
                                    
                    - 全文数据库/搜索引擎
                    - 学术圈社交网络
                    - 文献求助（论坛）
                    - 通过文章通讯作者邮箱索取
                    - 图书馆文献传递服务
                
                    文献检索的途径
                                
                      1.数据库（Web of Science最重要）
                                    
                      - 全文数据库：是收录文献全文的数据库，以sciencedirect、springer为代表
                      - 文摘数据库：是收录摘要、文献来源和文献引证关系的数据库，以三大索引数据库SC（科学文索引）、EI（工程索引）、ISTP(科技会议录索引）为代表，(三大索数据库统于ISI web of knowledge)
                    - 收费商业学术数据库
                    - 免费学术数据库（主要是学校数据库）
                
                    Web of Science数据库
                                        
                    (1) ISI Web of Science是全球最大、覆盖学科最多的综合性学术信息资源，收录了8700多种核心学术期刊。  
                    (2)强大的检索技术：主题检索、引文检索、化学结构检索、高级检索等，确保信息检索的全面与准确。  
                    (3)丰富而独特的引文信息：参考文献、施引文献、相关文献，揭示科学研究课题的过去、现在与将来。  
                    (4)以知识为基础的相关信息资源导航和链接：期刊目次、期肝刊影响因子、全文主题深度标引、馆藏、文献传递系统、专家库、OpenURL服务等，迅速获取全文。  
                    (5)Email定题服务，随时了解最新进展
                    (6)通过ISI Essential Science Indicators可以系统地、有针对性地分析国际科技文献综合全面的定量数据、清晰准确的统计以及与其它信息资源的有价值的连接。  
                    （7）在检索结果界面上，通过左上角的排序功能，您可以按照收录日期、被引用次数、相关度、第一作者姓名、来源期刊、出版年代等方式对检索结果重新排序。    
                
                    2.搜索引擎：谷歌学术、百度学术
                    - 共同特点：多种参考文献的导出格式；创建文献库(library)的功能；文献更新提醒功能
                    - 百度学术：免费下载链接；参考文献链接；学科分类功能
                    - 谷歌学术：检索量、相关性。镜像；翻墙
                
                    2.3文献追踪（Email Alert）
                    追踪某一个作者或某一篇文章  
                
                    3、学术社交网站
                    1.Google Scholar
                    - h指数：h-index又称为h指数或h因子（h-factor），是一种评价学术成就的新方法。 h代表"高引用次数"（high citations），一名科研人员的h指数是指他至多有h篇论文分别被引用了至少h次。 h指数能够比较准确地反映一个人的学术成就， 一个人的h指数越高，则表明他的论文影响力越大。一般博导的h指数要在10以上
                    - i10指数：某学者所发表的论文中引用次数超过10 次的个数
                
                    2.ResearchGate  
                    3.LinkedIn 领英   
                    4.MENDELEY   
                    5.RESEARCHERID      
                    6.ORCiD Open Researcherand Contributor ID'开放科研人员与贡献者的身份识别码 
                
                    4、文献管理
                    - Endnote
                    - Mendeley
                    - Reference Manager
                    - Zotero
                    - ReadCube
                    - NoteExpress（国产的）
                    # 二、论文阅读与分析
                    ## 1、科技论文介绍
                    ### 1.1论文类型
                    Research（研究型）、Review（综述）、Theoretical(理论)、Methodological（方法）、Case study（案例）
                    1.2研究型论文结构：传统AIMRaD结构
                    ![](https://api2.mubu.com/v3/document_image/83ca89ab-8fdf-4ff2-8a36-5d0bc11539b8-2968706.jpg)
                    - Abstract： Why they did it? How they did it? What did they find? What does it mean？
                    - Introduction： What is known about the topic? What is not known? What questions the authors asked and answered
                    - Methods： What experiments were done to answer the questions stated in the Introduction?
                    - Results： What have been found？
                    - Discussion： How the results support the conclusion？
                    - Conclusion： What did they find？ Better explain them？
                                    
                    ## 2、文献阅读
                    ### 2.1文献阅读概述
                    1.文献选择的原则
                    - 高质量文献：期刊影响因子、研究领域内代表性学者、试验室的文章
                    - 发表时间：新旧结合
                    - 选择文献由浅入深，逐步深入。
                    - 选择文献先多后少，由杂到精
                    - 选择文献先综述后论文，先中文后外文。（适用于入门者）
                    2.文献阅读的顺序
                    ![](https://api2.mubu.com/v3/document_image/9e6b0f1f-0266-4799-b7f9-4457b3b9f11f-2968706.jpg)
                    标题→摘要→结论→图表→引言→结果→讨论→方法
                    3.注意事项
                    （1）寻找课题阶段：重点读讨论和结论以及展望   
                    （2）课题设计阶段：主要是材料和方法。若只需了解一下该研究的思路：可选取摘要及引文与结论进行泛读。  
                    （3）自己熟悉的领域：细看摘要，略读前言，再看结果中的图表，最后读一下自己感兴趣的讨论部分。    
                    （4）批判性阅读：
                    - 由点到面，边看边查：有助于回忆与理解
                    - 集中时间看文献，做好记录和总结，定期复习
                    - 多数文章(约80%)，依据情况泛读
                    - 重要的文献(约20%)，全文精读
                    - 用怀疑的眼光去看文献
                    4.阅读笔记
                    （1）重要的文献一定要打印出来  
                    （2）记些什么？  
                    - 在文章题目的空白处写一段总结(研究主题、亮点、个人感受等)；
                    - 文献中的任何不足都应该记下来；
                    - 文献中好的词汇，句型；
